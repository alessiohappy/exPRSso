HOW TO AND EXAMPLE

Functionality

There are few operations to be carried out before to reproduce a PRS in your cohort, specifically:

  1. Get .ped and .map files

    2. These file can be computed using PLINK, the free open-source whole genome association analysis toolset.

    • v1.07 link: PLINK v1.07
    • v1.9/2.0 link: PLINK v1.9/2.0 beta

    Here's a brief description of the required structure for the .ped and .map files.

    • Pedigree file: a text file reporting pedigree information and genotype calls. A pedigree file is a white-space (space or tab) delimited file with extension .ped, has no header, each row represents a single individual, and is composed by these columns:
      1. Family ID (FID)
      2. Individual ID (IID)
      3. Paternal ID (PID)
      4. Maternal ID (MID)
      5. Sex
      6. Phenotype
      7. Alleles (variable number of columns)
    • Map file: is a text file that must contain 4 columns:
      1. Chromosome: a number from 1 to 22, X or Y
      2. SNP reference sequence: such as rsXXXXX
      3. Genetic distance
      4. BP position

    Before to upload .ped and .map files in exPRSso it is advisable to perform some quality controls (QCs) on your data. An example of data QCs is given here.

  2. Download the PRS summary

    3. The PRS summary can be gathered by different sources, such as PGScatalog. This is an online repository where PRSs for a wide range of traits can be found. Additional information are reported on their page.

    It is important that the PRS summary is uploaded as a text file without a header and with these columns: SNP, A1 and OR. Always check your PRS summary before to upload them in exPRSso.

  3. Upload your data

    4. Once you have collected the three required files, you can upload them in exPRSso. To do that, just go to the "exPRSso" section at the top of this page. You will be redirected to the application page. Here, you will find three upload buttons, one for each file to be uploaded.

  4. Compute the score

    5. Click on the "Calculate PRS" button and wait for the results to be ready!
