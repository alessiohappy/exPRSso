A Shiny application to assess the reproducibility of a Polygenic Risk Score (PRS).

## Overview

exPRSso is a user-friendly web application built with R-Shiny that allows to reproduce a PRS in your data, starting from your genetic data and a published PRS. The app provides an intuitive interface for uploading genetic data files and visualizing PRS results.

## Features

- Upload and process MAP and PED files
- Import a published PRS summary 
- Calculate weighted and unweighted PRS
- Interactive visualization of results
- Export results in CSV format
